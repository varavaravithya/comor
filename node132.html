Branch Hazard

รูป 6.35 แสดงชุดของคำสั่ง เมื่อมีคำสั่ง Branch ในไพพ์ลายน์ ซึ่งมีการ Fetch คำสั่งทุกวงรอบการทำงาน เนื่องจากการตัดสินใจของคำสั่ง Branch ทำในสเตท EX จึงเกิดปัญหา Control Hazard ขึ้น เนื่องจากการเกิด Branch Hazard เกิดน้อยกว่า Data Hazard มาก และมีความง่ายในการทำงานความเข้าใจปัญหาที่เกิดขึ้น เราจะพิจารณาวิธีการแก้ไขสองวิธี

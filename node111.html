การจัดการกับ Exception

ในการออกแบบโปรเซสเซอร์ จำเป็นต้องรองรับ Exception การเกิด Exception จะเกิดขึ้นในกรณีของการเกิด Overflow ในส่วนของการเกิด Interrupt จะเป็นการเกิดจากสิ่งเร้าภายนอกเช่น keyboard, mouse หรืออุปกรณ์อื่นๆ การเกิด Exception หรือ Interrupt โปรเซสเซอร์จะต้องหยุดทำงานโปรแกรมนั้นๆ และย้ายทำงานไปยัง Exception Program Counter (EPC) อีกวิธีหนึ่งทำโดยการเทคนิคของ Vector Interrupt ที่ระบบคำสั่งที่รองรับทำงานของ Interrupt ปรือ Exception นั้นๆ

ในโปรเซสเซอร์ที่พิจารณาในบทนี้ สามารถเกิด Exception ได้สองลักษณะ

  • Undefined Instruction
  • Arithmetic Overflow

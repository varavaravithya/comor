การคาดเดา Branch แบบพลวัตร

การสมมุติให้ Branch Not Taken ที่เป็นวิธีง่ายๆ แต่อาจทำงานได้ไม่ดีในกรณีที่ไพพ์ลายน์ลึกมากๆ ซึ่งมีวิธีหลายลักษณะที่จะคาดเดาผลของ Branch วิธีการหนึ่งคือการจำตำแหน่งของคำสั่ง Branch และผลการตัดสินใจครั้งก่อนหน้าเพื่อมาช่วยในการตัดสินใจในวงรอบปัจจุบัน ซึ่งวิธีการดังกล่าวเรียกว่า Dynamic Branch Prediction

วิธีการหนึ่งสามารถทำโดยการใช้ Branch Prediction Buffer หรือ Branch History Table ที่เป็นหน่วยความจำขนาดเล็กที่เก็บแอดเดรสของคำสั่ง Branch โดยใช้ส่วนล่างของแอดเดรสในการอ้างอิง และมีบิตที่ระบุว่า Branch จะ Take หรือไม่ ซึ่งการทำการคาดเดาอาจใช้หนึ่งบิตหรือ หลายบิตก็ได้ โดยในรูป 6.37 แสดงการคาดเดาแบบสองบิต

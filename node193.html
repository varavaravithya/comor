การส่งข้อมูลระหว่างอุปกรณ์กับหน่วยความจำ

เราสามารถใช้หน่วยประมวลผลในการส่งข้อมูลระหว่างอุปกรณ์กับหน่วยความจำโดยอาศัย polling กลไกทางเลือกอื่นที่ช่วยเพิ่มประสิทธิภาพคือการสร้างการส่งของ data interrupt driven ในกรณีนี้ ระบบปฏิบัติการจะยังคงส่งข้อมูลในจำนวนไบต์ขนาดเล็กติดต่อกับอุปกรณ์ แต่เพราะว่าการทำงานของ I/O เป็น interrupt driven ระบบปฏิบัติการจะทำงานอื่นได้ในขณะที่ข้อมูลกำลังอ่านหรือเขียนกับอุปกรณ์ เมื่อระบบปฏิบัติการสังเกตการ interrupt จากอุปกรณ์ มันจะอ่านสถานะเพื่อตรวจสอบความผิดพลาด ถ้าไม่มี ระบบปฏิบัติการสามารถส่งข้อมูลชิ้นต่อไปได้

แม้ว่า interrupt-driven I/O ช่วยแบ่งเบาภาระหน่วยประมวลผลจากการคอยสำหรับทุกเหตุการณ์ของ I/O แต่การใช้วิธีนี้ในการติดต่อกับ hard disk overhead ที่เกิดขึ้นก็ยังมากเกิน เนื่องจากมันสามารถผลาญการทำงานส่วนใหญ่ของหน่วยประมวลผล ในขณะที่ disk กำลังส่งข้อมูล สำหรับอุปกรณ์ bandwidth สูงเช่น hard disk การส่งข้อมูลประกอบด้วยส่วนหลักของ relatively large blocks ของข้อมูล (100 ถึง 1000 ไบต์) ดังนั้นนักออกแบบได้เกิดความคิดสร้างกลไกสำหรับลดภาระของหน่วยประมวลผล และให้ device controller ส่งข้อมูลโดยตรงกับหน่วยความจำโดยไม่ต้องใช้หน่วยประมวลผล กลไกนี้เรียกว่า direct memory access (DMA) กลไกการ interrupt ยังคงใช้อยู่โดยอุปกรณ์ติดต่อกับหน่วยประมวลผล แต่เพียงเฉพาะ I/O transfer ที่สมบูรณ์หรือเมื่อเกิดความผิดพลาดขึ้น

DMA ถูกรวมเข้าไปกับกลุ่มตัวควบคุมแบบพิเศษ และกลายเป็น bus master ในการติดต่อกับหน่วยความจำโดยตรง มีการทำงาน 3 ขั้นตอนคือ

  1. หน่วยประมวลผล set up DMA โดยการเตรียม ID ของอุปกรณ์, กระบวนการทำงานบนอุปกรณ์, แอดเดรสของหน่วยความจำที่ข้อมูลจะถูกส่งไป และจำนวนของไบต์ที่จะส่ง
  2. เมื่อเริ่มมีการส่งข้อมูล DMA จะเตรียมแอดเดรสของหน่วยความจำสำหรับอ่านหรือเขียน ถ้ามีการใช้มากกว่าหนึ่งเส้นทางบนบัส DMA จะสร้างแอดเดรสถัดไปและเริ่มส่งข้อมูล ด้วยกลไกนี้ทำให้ DMA สามารถส่งข้อมูลที่มีขนาดเป็นพันไบต์ได้โดยไม่ต้องพึ่งหน่วยประมวลผล DMA controller หลายตัวมีหน่วยความจำบางส่วนอยู่เพื่อให้สามารถทำงานอยู่ได้ขณะที่รอเป็น bus master
  3. เมื่อ DMA ส่งข้อมูลเรียบร้อย controller จะ interrupt หน่วยประมวลผล

แม้ว่า DMA จะถูกใช้เพื่อลดภาระของหน่วยประมวลผล แต่ถ้าหน่วยประมวลผลต้องการใช้หน่วยความจำก็จะเกิดการล่าช้า อันเนื่องจากหน่วยความจำถูกใช้สำหรับ DMA transfer การใช้ cache จะช่วยลดจำนวนครั้งของการเข้าถึงหน่วยความจำของหน่วยประมวลผลลง และ bandwidth ของหน่วยความจำจะมีพื้นที่ว่างพอสำหรับให้อุปกรณ์ I/O ใช้งาน

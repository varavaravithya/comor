สมมุติ Branch Not Taken

เราสามารถจัดการกับปัญหาของ Branch Hazard อย่างง่ายโดยการสมมุติให้ Branch Not Taken โดย การ Fetch คำสั่งต่อมาตามลำดับ ถ้าเกิด Branch Taken ก็จะทำการลบคำสั่งที่กำลังทำการ Decode อยู่ไป ซึ่งจะไม่ส่งผลกระทบใดๆ กับการทำงานเนื่องจากคำสั่งที่ตามมายังไม่แก้ไขสถานะของโปรเซสเซอร์ เพราะเพียงอ่านค่าจาดรีจีสเตอร์เท่านั้น ซึ่งทำได้โดยการเปลี่ยนสัญญาณควบคุมเป็น 0 ในสเตท ID ให้หมด หมายความว่าเราจะต้องสามารถ Flush คำสั่งทิ้งไปในสเตท IF, ID, และ EX ได้


Vara Varavithya 2005-08-18

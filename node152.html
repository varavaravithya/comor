ขนาดของ Block จะส่งผลต่อ Miss Rate ดังแสดงในรุป 7.7 จะเห็นว่าเมื่อเพิ่มขนาดของ Block แล้ว Miss Rate จะลดลง ในขณะที่ขนาดของ Block ขยายเพิ่มมากขึ้นทำให้ จำนวน Block ในแคชมีจำนวนน้อยลง ทำให้มีการแทนที่ลงใน Block ที่มีจำนวนจำกัด ส่งผลให้ Miss Rate มีค่าสูงขึ้น เมื่อเพิ่มขนาดของ Block มากเข้าจะก่อให้เกิดปัญหาอีกลักษณะหนึ่งคือ เวลาที่ใช้เมื่อเกิดการ Miss จะมาก เนื่องจากต้องทำการแทนที่ Block ขนาดใหญ่จากหน่วยความจำลำดับต่อไป มายัง Block ในแคช ซึ่งเวลาดังกล่าวสามารถลดลงได้โดยการใช้เทคนิค Early Start หรือ เทคนิค Requested Word First
